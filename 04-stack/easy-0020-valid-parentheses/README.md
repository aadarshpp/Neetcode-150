# 20. Valid Parentheses

## Topics
- String
- Stack

## Difficulty
Easy

## Complexity
- Time: O(n)
- Space: O(n)

## Approach
The function uses a **stack** to match brackets: push openings, pop on matching closings, return false if mismatched, and true if the stack is empty at the end.
