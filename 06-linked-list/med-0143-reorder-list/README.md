# 143. Reorder List

## Difficulty
Medium

## Complexity
- Time: O(n)
- Space: O(1)

## Approach
Find the middle(second) of the list using the slow and fast pointer method, split the list at the middle, reverse the second half, and then merge the two halves alternately.
