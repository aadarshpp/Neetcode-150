# 1. Two Sum

## Topics
- Array
- Hash Table

## Difficulty
Easy

## Complexity
- Time: O(n)
- Space: O(n)

## Approach
Iterate through the array using the index. For each number, check if target - current number exists in the map. If it does, return the pair of indices. Otherwise, store the current number and its index in the map.
