# 217. Contains Duplicate

## Topics
- Array
- Hash Table
- Sorting

## Difficulty
Easy

## Complexity
- Time: O(n)
- Space: O(n)

## Approach
Hash each value as you see it. If it is already in the set, it is a duplicate. Otherwise, add it to the set.
