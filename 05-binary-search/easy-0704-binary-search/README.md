# 704. Binary Search

## Topics
- Array
- Binary Search

## Difficulty
Easy

## Complexity
- Time: O(log(n))
- Space: O(1)

## Approach
Given a sorted array, the target is either in the middle, on the left, on the right, or not present. This can be decided in O(1) time at each step, giving a total time complexity of O(log n) since we cut the search area in half each time.
